# Instant Messaging App build with React JS & Firebase

# Live Site : https://instant-messaging-app-ia.web.app/

## Technologies Used

1. React Js
2. React Router
3. FireBase Authentication
4. FireBase Database
4. JWT Auth Token 
4. Material UI
6. ServerLess

## Features:

1. Authentication
2. Secured API
3. Instant Messeging
4. Create User or Create Chat Room
6. Custom built API
7. Responsive
